#docker-backend-workspace

## Docker for Backend
- nginx
- php
- mariadb
- phpmyadmin
- redis
- redis-commander
---
## Docker Command
1. Clone project
```
git clone https://github.com/Thammasok/docker-backend-workspace.git
cd docker-backend-workspace/docker
```
2. Run Docker Compose
```
docker-compose up -d
docker-compose down -v
```
